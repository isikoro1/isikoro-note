## 概要

Python は、文法が比較的読みやすく、短いコードで処理を書きやすいプログラミング言語です。
Webアプリ、データ処理、業務自動化、AI・機械学習、スクリプト処理など、幅広い用途で使われます。

## 内容

Python を学ぶときは、最初からAIや機械学習に飛びつくより、まずは普通のプログラムとして読める・書ける状態にするのが重要です。

見る順番は次のようにすると分かりやすいです。

1. 変数、条件分岐、繰り返し
2. リスト、辞書、タプル、集合
3. 関数
4. ファイル読み書き
5. 例外処理
6. モジュールとパッケージ
7. 仮想環境
8. Web API
9. 自動化スクリプト
10. AI・データ処理

## Javaと比べたときの違い

Java は型やクラス構造を意識して、大きめの業務システムを堅く作る場面でよく使われます。
一方で Python は、短い処理を素早く書く、自動化する、データを加工する、といった用途に向いています。

ただし、Python でも業務で使うなら、型ヒント、例外処理、テスト、依存関係管理を軽視すると保守しづらくなります。

## 学習で見る観点

- 文法を暗記するより、処理の流れを読めるか
- リストや辞書を使ってデータを扱えるか
- ファイルやAPIからデータを取り込めるか
- 小さな自動化を書けるか
- AIに書かせたコードをレビューできるか
