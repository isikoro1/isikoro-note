## ざっくり

Servlet は、Java で HTTP リクエストを受け取り、HTTP レスポンスを返すための仕組みです。

## 何に使う？

Java の Web アプリケーションの土台として使われます。
フレームワークを使う場合でも、内部では Servlet の考え方が関係することがあります。
